SysPilot is a local web application designed to give you convenient and secure control over your machine, directly from any device on your local network. It transforms your browser into a powerful remote control for key operating system operations, multimedia management, and basic monitoring.

Built with Python (Flask) on the backend and a modern frontend interface (HTML, CSS, JavaScript), SysPilot offers a lightweight and efficient solution for Linux users.

How Does It Work?

SysPilot is installed as a service on your Linux machine. Once running, it deploys a web interface that you can access via any web browser (on a computer, tablet, or mobile phone) as long as you are connected to the same local network.

You do not need an external internet connection or cloud services. All control is performed privately and securely within your home or office network.

Power Control

Remotely shut down, restart, or lock your session with a single click, without needing to be physically in front of your machine.

Multimedia Management

Control media playback (play/pause, next, previous) and adjust system volume or mute it directly from the web.

Basic Monitoring

Get a quick overview of CPU and RAM usage, and your system's uptime, keeping you informed about its status.

User Administration

Manage users, roles, and permissions directly from the dashboard, controlling who has access to which features.

Customizable Commands

Adapt system actions to your needs by configuring the shell commands executed for each action.
